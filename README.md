# Online Supplementary Material Mulder, Voelkle, and Hamaker (under review)
Online supplementary materials for The Problem of Temporal Misalignment in Longitudinal Causal Research by Mulder, Voelkle, and Hamaker (under review). 

- `20260119_Research_TimeAggregation_Revised.rmd` Main file from which all analyses are run.
- `./R` Folder which contains all additional R functions used in the analyses.
